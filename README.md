# 📝 To-Do List App

A sleek and efficient **To-Do List application** built with modern web technologies. Stay organized, productive, and never forget a task again!

## ✨ Features

- ✅ **Add Tasks Easily** - Create new tasks with a simple interface
- 🗑️ **Delete Tasks** - Remove completed or unwanted tasks instantly
- 💾 **Persistent Storage** - All tasks are saved in localStorage (no data loss on refresh)
- 📱 **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- 🎨 **Clean UI** - Minimalist and intuitive user interface
- ⚡ **Lightning Fast** - No backend required, runs entirely in your browser

## 🚀 Quick Start

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Git (optional, for cloning)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Vishal-git123/To-do-list.git
   cd To-do-list
   ```

2. **Open the app:**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Python 2
     python -m SimpleHTTPServer 8000
     
     # Node.js (if you have http-server installed)
     npx http-server
     ```

3. **Start organizing:**
   - Type your task in the input field
   - Click "Add" or press Enter
   - Check off completed tasks
   - Click delete to remove tasks

## 💡 How It Works

1. **Add a Task** - Enter text and submit to add it to your list
2. **View Tasks** - All tasks are displayed with delete buttons
3. **Delete a Task** - Click the delete button to remove a task
4. **Automatic Saving** - Tasks are automatically saved to browser's localStorage
5. **Persistence** - Close and reopen the app, your tasks will still be there!
